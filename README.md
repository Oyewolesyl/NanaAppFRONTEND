# NanaAppFRONTEND

## Project Overview
Frontend of Nana, mobile app interface for children, parents, and doctors to track and visualize pain.

### Features
- Interactive body map for pain
- Pain history and timeline
- Multi-child support
- Notes/comments input
- Doctor view and feedback

### Tech Stack
- React Native
- Expo / React Native CLI
- Axios for API calls
- React Navigation
- Redux / Context API

### Branches
- main (production-ready)
- develop (active development)
- staging (testing/integration)
- feature/<name> (new features)
- fix/<name> (bug fixes)

### Setup
1. Clone repo
2. Install dependencies: `npm install`
3. Start development server: `npm start`
